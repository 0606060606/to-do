# TO DO LIST
用 Vue.js 3 建立的簡單TO DO LIST。user可以創建、更新和刪除待辦事項，並將代辦事項排序顯示。

# 功能
-可新增具有分類（study、Personal或Sport）的待辦事項
-使用顏色區分各類代辦事項
-標記待辦事項為完成/未完成
-編輯待辦事項的內容
-刪除待辦事項
-自動按創建時間排序待辦事項

# 運行
1.npm run dev
2.把終端機輸出的網址如(http://localhost:3000)複製到瀏覽器
